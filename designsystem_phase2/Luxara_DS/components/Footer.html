Footer Component

The Footer component provides essential information and navigation links at the bottom of a webpage. It is designed to be fully responsive, adapting its layout to different screen sizes while maintaining readability and usability.

Our Footer includes copyright information, a brief tagline, and social media links.

Code Snippet

Below is the HTML structure for a responsive Footer component. This example uses Bootstrap's grid system to adapt content for different screen sizes.


<footer class="bg-dark text-white py-3">
  <div class="container">
    <div class="row align-items-center">
      <div class="col-md-4 text-center text-md-start">
        <p class="mb-0">© 2025 Your Brand | All rights reserved</p>
      </div>
      <div class="col-md-4 text-center d-none d-md-block">
        <p class="mb-0">Your tagline here.</p>
      </div>
      <div class="col-md-4 text-center text-md-end">
        <ul class="list-inline mb-0">
          <li class="list-inline-item"><a href="#" class="text-white">Link 1</a></li>
          <li class="list-inline-item"><a href="#" class="text-white">Link 2</a></li>
          <li class="list-inline-item"><a href="#" class="text-white">Link 3</a></li>
        </ul>
      </div>
    </div>
  </div>
</footer>

Usage Considerations

  • Content Hierarchy: Prioritize important information such as copyright, and then add secondary content like taglines or social links.
  • Responsive Behavior: Utilize Bootstrap's responsive utility classes to ensure content rearranges gracefully on smaller screens.
  • Consistency: Maintain consistent branding and visual style with the rest of the website.

Accessibility Considerations

  • Color Contrast: Ensure sufficient color contrast between text and background for readability.
  • Keyboard Navigation: All interactive elements (links, buttons) should be navigable and usable via keyboard.
