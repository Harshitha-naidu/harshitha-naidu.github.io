Buttons

Buttons are essential interactive elements that allow users to perform actions within the system. They are designed to be intuitive and provide clear feedback upon interaction.

Accessibility Considerations

  • Ensure buttons are keyboard navigable and can be activated with the Enter or Space key.
  • Provide clear and concise `aria-label` attributes for buttons where the text content alone might not be sufficiently descriptive for screen reader users.
  • Maintain a minimum target size of 44x44 CSS pixels for interactive elements to ensure easy activation on touch devices.

Usage

  • Use buttons for primary actions like submitting forms, navigating, or triggering specific functions.
  • Avoid using buttons for purely decorative purposes.
  • Ensure consistent styling and behavior across all buttons in the design system.
