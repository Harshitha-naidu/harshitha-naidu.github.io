Cards Component

Cards are versatile content containers that can display a variety of information, such as images, text, and links. They are used to group related content in a visually appealing and organized manner.

This component is designed to be responsive, adapting its layout to different screen sizes using Bootstrap's grid system. We will showcase five different card layouts:

  • Vertical Card: Image on top, content below (default for mobile, can be used for larger screens too).
  • Horizontal Card (Image Left): Image on the left, content on the right (for tablet and desktop views).
  • Horizontal Card (Image Right): Content on the left, image on the right (for tablet and desktop views).

Code Snippet (Horizontal Card - Image Left)

Below is the HTML structure for a horizontal card component with the image on the left.


<div class="col-12">
  <div class="card">
    <div class="row g-0">
      <div class="col-md-4">
        <img src="../assets/card_image.jpg" class="img-fluid rounded-start" alt="Descriptive image text">
      </div>
      <div class="col-md-8">
        <div class="card-body">
          <h3 class="card-title">Card Title</h3>
          <p class="card-text">This is a wider card with supporting text below as a natural lead-in to additional content. This content is a little bit longer.</p>
          <a href="#" class="btn btn-primary">Read More</a>
        </div>
      </div>
    </div>
  </div>
</div>

Usage Considerations

  • Choose between vertical and horizontal layouts based on content type and screen size. Vertical cards work best for image-heavy content on mobile, while horizontal cards are ideal for larger screens.
  • Maintain a balanced ratio between images and text content to ensure neither element dominates the card's visual hierarchy.
  • Use Bootstrap's responsive grid classes to ensure cards adapt smoothly across different screen sizes.

Accessibility Considerations

  • Always provide meaningful alt text for images to support screen reader users
  • Ensure all interactive elements are keyboard accessible
  • Maintain proper image sizing and aspect ratios
